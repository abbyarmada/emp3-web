440    /**
441     * This methods places the map into freehand draw mode if no exception is raised. The user can use
442     * drag gestures to draw lines on the map.<br/>
443     *<br/>
444     * When entering freehand mode:<br/>
445     * 1) - The map is locked in smart lock mode.<br/>
446     * 2) - A MAP_ENTERED_FREEHAND_DRAW_MODE event is generated.<br/>
447     * 3) - At the first drag gesture a MAP_FREEHAND_LINE_DRAW_START is generated when the first N
448     * points (N > 1) have been drawn. As the drawing takes place a line with the specified style is
449     * displayed on the map.<br/>
450     * 4) - A MAP_FREEHAND_LINE_DRAW_UPDATE event is generated as the user continues the drag gesture.<br/>
451     * 5) - When the user ends the drag gesture (lifts the finger or stylist), a MAP_FREEHAND_LINE_DRAW_END
452     * event is generated. The event shall contain a list of geospatial coordinates of the line drawn.
453     * Once the event return the line is removed from the map, and the list of coordinates are deleted.<br/>
454     * 6) - The user may choose to initiate a new drag gesture to generate a new line therefore returning
455     * to step # 3.<br/>
456     * 7) - When drawComplete is called a MAP_EXIT_FREEHAND_DRAW_MODE is generated, the map is unlocked,
457     * 8) - EditorMode changes to FREEHAND_MODE
458     * and the map exits freehand draw mode.<br/>
459     * @param initialStyle This parameters contains the initial style to use for the draw. A value of null causes an EMP_Exception.
460     * @param listener This parameters is an event listener to handle the freehand draw events.
461     * @throws EMP_Exception if the map does not support freehand drawing, or there is a draw or edit in progress,
462     *     map is MapMotionLockMode is LOCKED or SMART_LOCK.
463     */
464    void drawFreehand(IGeoStrokeStyle initialStyle, IFreehandEventListener listener)
465            throws EMP_Exception;
